# Sky World 🌤️

تطبيق طقس ذكي لنظام أندرويد، يعرض حالة الطقس بدقة ويربطها مباشرة بالأنشطة اليومية (مشي، ركض، دراجة، قيادة، طيران)، بواجهة عربية بالكامل وتصميم بصري مخصص.
